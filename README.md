# Iris Flower Classification

## Overview
This project classifies Iris flowers into Setosa, Versicolor and Virginica using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features
- Data Preprocessing
- Model Training
- Prediction
- Accuracy Evaluation

## Dataset
Iris Dataset
